gulp开发项目的详细教程

Share

gulp（官网）是一种自动编译项目的任务工作流，是一种以任务为单位，在node环境下进行执行的工作流，借助一些插件可以帮助开发者进行less的编译、提取htnl公共部分、图片压缩、文件合并、自动刷新浏览器等工作

(一)在全局安装

  • 如果没有安装node.js的需要先下载安装node.js
  • 打开cdm命令面板，在全局安装gulp-cli

在项目目录文件安装

  1. 在项目的根目录下初始package.json文件，用来记录项目的一些信息

  2. 把 gulp 安装到本地项目

注：在用npm安装第三方包的时候，执行--save和--save-dev命令，是将第三方包的信息记录在package.josn文件中，便于其他开发人员查阅，--save是存储项目中需要用到的包，如jquery；--save-dev是存储项目中的一些开发命令包，不需要引入到页面中去

gulp中用的api

前端必推荐书籍

Share

作为一名程序猿，每日除了与代码打交道之外，上网查阅最新的资料，一些书籍也是我们必读的，这些书籍是一些牛人总结下来的经验，如果我们能够将其化为己用的话，对我们将会有很大的帮助，因此我整理了一些前端的书籍，以供大家学习交流

入门书籍

这里推荐一些入门的书籍，对前端的代码，书写等有一些了解

  • 首推w3c课程网站
  • 《Head First Html 与 Css、XHTML》

    妙趣横生的一本入门css的书，相信会让你对css感兴趣

  • 《HTML5 与 CSS3 基础教程（第八版）

    非常适合初学前端的人看，浅显易懂，有配套代码！

  • 《CSS权威指南（第三版）》

    这本就是进阶了，系统全面地讲解了 CSS 规范，通读之后相信网页特效都不在话下。

  • 《CSS那些事儿》

    全书以传达CSS布局思维为中心，通过页面中的文字、图片、表格、表单等常见元素的处理及各种页面布局方式的使用，使读者能深入了解到如何在页面中更好地运用CSS布局

  • 《JavaScript DOM编程艺术》

    一开始如果对JavaScript的晦涩难懂的语法概念实在提不起兴趣，可以先把这本书的例子敲一遍，让你体会到开发网页的乐趣

  • 《JavaScript高级程序设计》

    红宝书虽然号称高级，但其实是帮助入门的，Zakas大神把概念深入浅出，将JavaScript的语法要点讲清楚明白。小红书配合犀牛书，相互印证，谁读谁知道。

进阶书籍

通过上面的入门书籍，你会对css、html、js有了初步的了解，但是这远远满足不了你的学习和工作需要，因此，你需要读一些进阶的书籍，来提升自己的能力。

  • 《CSS揭秘》

    这本书讲了很多CSS的技巧，看完之后你会拍着脑袋说原来CSS还有这么多你不知道的东西。

  • 《JavaScript权威指南》第6版

    是学习核心JavaScript语言和由Web浏览器定义的JavaScript API的指南和综合参考手册。

  • 《JavaScript设计模式》

    进阶教程，得有一定功底才能看懂。反之，如果你轻而易举就能拿下此书，说明你已非等闲之辈

  • 你不知道的JavaScript》

    这本绝对是神书，让你了解JavaScript不为人知的另一面，把闭包、异步这些讲得很通透。

  • 《ECMAScript6入门》

    现在是ES6的时代了，可以看看阮老师的这本入门，领悟ES6的强大。

  • 《Javascript语言精粹》

    这本书讲解了js中的一些精髓之处，指出js中的一些坑，可以让你在开发的时候避免这些错误

  • 《JavaScript忍者秘籍》

    这本书是jQuery库创始人编写的一本深入剖析JavaScript语言的书 对JavaScript语言的重要部分（函数、闭包和原型）进行深入、全面的介绍，以及讲述跨浏览器代码如何编写

  • 《高性能 JavaScript》

    各种优化js的手段

前端其他的书籍

这些书和前端有关，学习之后让你在开发的时候更能得心应手

  • 《锋利的jQuery》第二版

    循序渐进地对jQuery的各种函数和方法调用进行了介绍，对jQuery分析的逻辑非常值得学习。

  • 高性能网站建设指南》

    前端开发可以优化网站剩余70%~80%性能，这本书不算厚，几个小时就能看完，推荐前端开发工程师看看。

  • 《Web开发敏捷之道》第二版

    想成为优秀的前端工程师，强烈建议阅读此书！
